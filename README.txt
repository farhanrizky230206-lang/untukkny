WEBSITE CONFESSION — VERSI SLIDE
=================================

Website sudah siap untuk Netlify dan Vercel karena file utamanya bernama index.html.

PENTING: GANTI NOMOR WHATSAPP
1. Buka app.js.
2. Di baris atas cari:
   const WHATSAPP_NUMBER = "6281234567890";
3. Ganti dengan nomor kamu memakai format internasional.
   Contoh:
   081234567890
   menjadi:
   6281234567890

JAWABAN WEBSITE
- Tombol "Iya, aku bersedia" membuka WhatsApp dengan jawaban positif.
- Tombol "Aku butuh waktu" membuka WhatsApp dengan jawaban butuh waktu.
- Dia tetap perlu menekan tombol Kirim di WhatsApp. Website tidak boleh mengirim pesan diam-diam.

FOTO
Ganti file berikut tanpa mengubah namanya:
assets/images/foto-1.jpg
assets/images/foto-2.jpg
assets/images/foto-3.jpg
assets/images/foto-4.jpg
assets/images/foto-5.jpg

MUSIK
Masukkan file ke:
assets/music/our-song.mpeg
atau:
assets/music/our-song.mp3

Website sudah menerima kedua ekstensi tersebut.

DEPLOY NETLIFY
Ekstrak ZIP, lalu drag folder romantic-confession-slides ke Netlify Drop.

DEPLOY VERCEL
Upload folder ke GitHub, lalu Import Project di Vercel.
Framework preset: Other
Build command: kosong
Output directory: kosong
